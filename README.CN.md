
# H-toolkit

[中文](./README.CN.md)

基于 Wails 的 Multiplatform 桌面端工具箱。

## 功能
- **JSON工具**:支持Json格式化处理、压缩、转义、添加转义等功能。
- **时间戳工具**:支持当前时间戳获取、时间戳与日期时间的相互转换。

## 技术栈
- **语言**: Vue 3 + TS 5 + Golang
- **UI 框架**: TDesign
- **构建工具**: Wails

## 开发指南

wails命令行开发:

```bash
wails dev
```

## 构建

```bash
wails build
```
